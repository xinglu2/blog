    • HTML是谁发明的
    HTML是由一名叫Tim Berners-Lee的科学家发明的。他发明HTML的目的，是为了方便不同大学的科学家们可以更容易地获取彼此的研究文档。
    编辑于2021.4
    HTML 起手应该写什么
    < !DOCTYPE html>
    < html lang="zh-CN">
    <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <title>Document</title>
    </head>
    <body>
    </html>
    编辑于2021.4
    常用的表章节的标签
    <h1>~<h6>：这是六个不同的标题标签;
    <section>：标签定义文档中的节（section、区段）。
    <article>：规定独立的自包含内容。
    <main>：规定文档的主要内容。
    <aside>：定义其所处内容之外的内容。
    编辑于2021.4
    全局属性
    class：规定元素的一个或多个类名（引用样式表中的类）。
    contenteditable：使任何元素都可以被编辑。
    hidden：可以让任何东西看不见。
    id：不到万不得已不要用，因为它不报错。
    style：为 HTML 文档定义样式信息。
    title：鼠标浮在上面时显示的内容。
    tabindex: 取值是整数，数越大越最后访问，但是值为0时会最后一个访问到，值为-1时，永远访问不到。
    编辑于2021.4
    常用的内容标签
    <code>： 定义计算机代码文本。
    <em>：定义强调文本。强调语气
    <strong>：定义语气更为强烈的强调文本。强调本身
    <pre>：定义预格式文本。
    编辑于2021.4

    心得

    HTML标签有很多，每一种标签有不同的含义，但是有很多标签作用又十分的相似，比如说<em>和<strong>，它们都用来表示强调，但是细节之处又不一样， <em>是强调语气，而<strong>是本身就很重要。类似的标签还有很多，需要我们自己去摸索，在学习HTML时，我们要学习标签中特殊的地方，这样会使我们记得更加牢固也不容易混淆。
